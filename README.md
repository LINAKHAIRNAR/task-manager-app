# Task Manager UI

A simple task manager user interface built using HTML and CSS.

## Features
- Add task section
- Task cards
- Checkbox completion effect
- Remove buttons
- Hover and click animations

## Technologies Used
- HTML
- CSS

## What I Learned
- Flexbox layouts
- UI structuring
- Interactive styling
- State-based CSS styling
