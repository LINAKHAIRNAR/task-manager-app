Task Manager

A simple Task Manager web application built using HTML, CSS, and JavaScript. Users can add tasks, mark them as completed, and remove tasks dynamically.

Features

- Add new tasks
- Remove tasks
- Mark tasks as completed using checkboxes
- Dynamic task creation with JavaScript
- Hover and click animations
- Responsive and clean user interface

Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

What I Learned

- DOM manipulation
- Event handling
- Dynamically creating and removing elements
- Flexbox layouts
- UI structuring
- Interactive styling
- State-based CSS styling
